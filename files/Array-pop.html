pop

Синтаксис

arrayObj.pop()

Аргументы

Описание, примеры

Этот метод изменяет исходный массив.

Пример:
Выделить все 
myFish = ["angel", "clown", "mandarin", "surgeon"];
popped = myFish.pop();
 
// теперь popped = "surgeon"
// myFish = ["angel", "clown", "mandarin"]
