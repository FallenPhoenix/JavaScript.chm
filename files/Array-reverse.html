reverse

Синтаксис

arrayObj.reverse()

Аргументы

Описание, примеры

Данный метод изменяет исходный массив.
После его применения порядок элементов в массиве меняется на обратный.

Он возвращает измененный массив.

Пример:
Выделить все 
arr = [1,2,3]
 
a = arr.reverse() // [ 3, 2, 1]
 
alert(arr == a) // true
