getTime

Синтаксис

getTime()

Аргументы

Описание, примеры

Значение, возвращаемое методом getTime, равно количеству миллисекунд, прошедших с полуночи 1 января 1970 года. Этот метод можно использовать для присвоения значения другому объекту Date.

Этот метод функционально эквивалентен методу valueOf.

Пример: Копируем значение
// Копируем значение theBigDay в sameAsBigDay:
theBigDay = new Date("July 1, 1999")
sameAsBigDay = new Date()
sameAsBigDay.setTime(theBigDay.getTime())
