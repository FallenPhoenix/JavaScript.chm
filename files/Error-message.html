message

Читаемое описание ошибки

Описание, примеры

Это свойство содержит краткое описание ошибки. Как правило, это основной источник информации о произошедшей ошибке.

Пример: 
var e = new Error("Произошла проблема"); 
// e.message = Произошла проблема
throw e;
