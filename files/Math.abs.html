Math.abs

Синтаксис

var abs = Math.abs(x);

Аргументы

x
числовой аргумент

Описание, примеры

Передача нечисловой строки или undefined/пустой переменной вернет NaN. Передача null вернет 0.

Пример:
Выделить все 
Math.abs('-1');     // 1
Math.abs(-2);       // 2
Math.abs(null);     // 0
Math.abs("string"); // NaN
Math.abs();         // NaN
