toExponential

Синтаксис

number.toExponential( [fractionDigits] )

Аргументы

fractionDigits
Количество цифр после запятой. По умолчанию - без округлений

Описание, примеры

Число в экспоненциальной записи возвращается в виде строки с одной цифрой до десятичной точки и округляется до fractionDigits цифр после десятичной точки.

Если аргумент fractionDigits не указан, то берется столько цифр после десятичной точки, сколько необходимо.

Если число содержит больше цифр, чем указавно в параметре fractionDigits, число округляется до ближайшего, состоящего из fractionDigits цифр, аналогично методу toFixed.

Пример:
Выделить все 
var num=77.1234;
 
alert(num.toExponential()) // 7.71234e+1
alert(num.toExponential(4)) // 7.7123e+1
alert(num.toExponential(2)) // 7.71e+1
