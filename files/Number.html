Number

Создание

numObj = new Number(value) // не используется
numObj = Number(value)
// обычно используются статические методы Number.*

Аргументы

value
значение для преобразование в тип Number

Описание, примеры

При необходимости, javascript сам создает Number объекты из числовых значений. Создавать объекты Number в явном виде требуется редко.

Основное назначение Number объектов состоит в сборе свойств в один объект, а также в преобразовании к числу вызовом Number(value).

Методы

toFixed
Округляет число до фиксированного количество цифр после запятой. Возвращает строку.
toPrecision
Округляет число до указанного общего количества цифр
toExponential
Возвращает строку с записю числа в экспоненциальном виде
toString
Возвращает строковое представление объекта.

Свойства

Number.MAX_VALUE
Возвращает наибольшее число, которое может быть представлено в яваскрипт. Примерно равно 1.79E+308.
Number.MIN_VALUE
Возвращает ближайшее к нулю число, представимое в яваскрипт. Примерно равно 5.00E-324.
Number.NaN
Особое значение Not-A-Number, которое указывает, что арифметическое выражение возвратило значение, не являющееся числом.
Number.NEGATIVE_INFINITY
Возвращает значение, которое меньше любого отрицательного числа, которое может быть представлено в яваскрипт.
Number.POSITIVE_INFINITY
Возвращает значение, большее любого числа, которое может быть представлено в яваскрипт
