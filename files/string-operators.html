Строковые операторы

Описание, примеры

Оператор +, примененный к операндам, один из которых строка, является оператором конкатенации (сложения) строк.

Пример: 
"my" + "string" == "mystring"
"a" + 1 = "a1"
1 + "a" = "1a"

Оператор += является сокращенным вариантом записи:

s += a // то же что и 
str = str + a
