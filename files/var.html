var

Объявить переменную (или несколько) в текущей области видимости

Синтаксис

var varname1 [= value1 [, varname2 [, varname3 ... [, varnameN]]]];

Аргументы

varnameN
Имя для переменной
valueN
Начальное значение переменной, любое выражение

Описание, примеры

Область видимости переменной - текущая функция, или глобальная, если переменная объявлена вне функций.

В javascript блок не задает область видимости.

Переменная внешней функции видна во внутренней, благодаря наличию замыканий.

Пример: 
var num_hits = 0, cust_no = 0
var t
