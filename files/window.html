window

Объект window сочетает два в одном: глобальный объект javascript и окно браузера.

Для обращения к функциям и методам window не нужно указывать объект:

window.decodeURI(..)
// то же что и
decodeURI(..)

Вообще, любая переменная в конечном счете(если не найдена локально) ищется в глобальном объекте.

// можно указать window явно - будет работать:
a = new window.String("test")

Да, и кстати window является свойством window:

xhr = new window.window.window.XMLHttpRequest()

Кроме роли "глобального объекта", window также предоставляет интерфейс для работы с окном браузера.

Методы

window.close
Закрыть окно браузера
window.open
Открыть новое окно браузера и загрузить в него указанный ресурс
window.focus
Сделать вызвавшее окно активным

Свойства

window.location
Получает/устанавливает URL окна и его компоненты
window.parent
Родительское окно в иерархии фреймов
window.top
Ссылка на верхнее окно в иерархии фреймов
window.name
имя окна
window.frames
список фреймов текущего окна, включая iframe
window.opener
окно, открывшее текущее вызовом window.open
window.closed
Закрыто ли данное окно, используется с window.open
window.status
Позволяет устанавливать и получать статусную строку окна
